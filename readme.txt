A WP User Manager add-on for the [Newsletter plugin](https://wordpress.org/plugins/newsletter/) to add a subscription checkbox to the WP User Manager registration form.

== Description ==

This addon integrates WP User Manager with the [Newsletter](https://wordpress.org/plugins/newsletter/) plugin along with the [Newsletter WP Users Integration free addon](https://www.thenewsletterplugin.com/documentation/addons/extended-features/wpusers-extension/).

= WP User Manager plugin is required. =

> This is a free add-on for the [WP User Manager plugin](https://wpusermanager.com). You must download and install the [WP User Manager plugin](https://wordpress.org/plugins/wp-user-manager/) before you can use this addon.
